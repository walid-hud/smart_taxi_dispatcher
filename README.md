# SAS project at [ENAA](https://enaa.ma)

### How to run
```sh
git clone https://github.com/walid-hud/smart_taxi_dispatcher.git
cd ./smart_taxi_dispatcher
npm i 
node ./dist/index.js
```

I used typescript but you can find the js version [here](https://github.com/walid-hud/smart_taxi_dispatcher/blob/main/dist/index.js)
